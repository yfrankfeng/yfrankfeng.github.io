How to use blacklight hierarchy gem to implement hierarchical facets

Blacklight hierarchy provides a light way to support hierarchical facets,
but the documentation is a bit hard to understand, I just write down what I have done to get a working prototype.

Update Gemfile

Add this line to Gemfile:

and run:

Define facets in catalog_controller

Edit app/controllers/catalog_controller.rb:

config.add_facet_field solr_name('summary_general_research_area', :facetable),
                       label: "Topic",
                       :partial => 'blacklight/hierarchy/facet_hierarchy'

config.facet_display = {
    :hierarchy => {
        'summary_general_research_area' => [['sim'],':']
    }
}

NB: the facet field name in solr is concatenated from above ‘summary_general_research_area’ and ‘sim’,
e.g. the solr facet field name is: ‘facet-summary_general_research_area_sim’

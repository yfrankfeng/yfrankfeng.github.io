Use ReactJS to build IRIS statistics app

Setup project

npx create-react-app iris-statistics
cd iris-statistics
yarn start

You will see the UI:

https://snipboard.io/FawR2z.jpg

Component

Add LanguageList.js:

import React from 'react';

class LanguageList extends React.Component {
    render() {
        return (
            <div className="language-list">
                <h1>My favorite language list</h1>
                <ul>
                    <li>Python</li>
                    <li>NodeJS</li>
                    <li>Java</li>
                </ul>
            </div>
        );
    }
}

export default LanguageList;

Use it in App.js:

import LanguageList from './LanguageList';

function App() {
  return (
    <div className="App">
      <header className="App-header">
        <p>
          <LanguageList />
        </p>
      </header>
    </div>
  );
}

import React from 'react';

import Square from './Square';

class Board extends React.Component {
    renderSquare(i) {
        return <Square />;
    }

    render() {
        const status = 'Next player: X';

        return (
            <div>
                <div className="status">{status}</div>
                <div className="board-row">
                    {this.renderSquare(0)}
                    {this.renderSquare(1)}
                    {this.renderSquare(2)}

                <div className="board-row">
                    {this.renderSquare(3)}

                <div className="board-row">
                    {this.renderSquare(6)}
                    {this.renderSquare(7)}

}

export default Board;

Making interactive component

Test onClick event, in Square.js:

<button className="square" onClick={function() { alert('click'); }}>
    {this.props.value}
</button>

Or with arrow function:

<button className="square" onClick={() => alert('click') }>

Add constructor to Square:

constructor(props) {
    super(props);
    this.state = {
       value: null,
    };
}

and:

<button className="square" onClick={() => this.setState({value: 'X'}) }>
   {this.state.value}
</button>

Complete the game

To collect data from multiple children, or to have two child components communicate with each other, you need to declare the shared state in their parent component instead. The parent component can pass the state back down to the children by using props; this keeps the child components in sync with each other and with the parent component.

Add a constructor to Board:

constructor(props) {
    super(props);
    this.state = {
      squares: Array(9).fill(null),
    };
}

Change renderSquare:

renderSquare(i) {
    return <Square value={this.state.squares[i]} />;
}

Locate the following string to continue

After these changes, we’re again able to click on the Squares to fill them, the same as we had before. However, now the state is stored in the Board component instead of the individual Square components. When the Board’s state changes, the Square components re-render automatically. Keeping the state of all squares in the Board component will allow it to determine the winner in the future.

Hyrax

Hyrax UI customization

Header

Top bar for language/login

In original Hyrax, remove this line in app/views/layouts/hyrax.html.erb:

<%= render '/masthead' %>

If using a customized layout, e.g. app/views/layouts/ncelphome.html.erb, doing the same thing there.

Banner

Edit file: app/views/layouts/homepage.html.erb (copied form Hyrax):

<% content_for(:navbar) do %>
  <div class="image-masthead">
    <div class="background-container">
      <%= image_tag(t('ncelp.banner_image'), alt: "NCELP") %>

Footer

Edit app/views/shared/_footer.html.erb:

<footer class="navbar navbar-inverse site-footer ncelp-footer-border">
  <div class="container-fluid row">
    <div class="col-sm-4 ncelp_footer_container" style="padding-top: 20px;">
      <%= image_tag(t('ncelp.footer.ncelp_logo'), :class=>'ncelp_footer_image', alt: "NCELP") %>
    </div>
    <div class="col-sm-2 ncelp_footer_container">
      <%= image_tag(t('ncelp.footer.uoy_logo'), :class=>'ncelp_footer_image', alt: "University of York") %>
    </div>
    <div class="col-sm-6 ncelp_footer_container">
      <%= image_tag(t('ncelp.footer.education_dept_logo'), :class=>'ncelp_footer_image', alt: "Department for Education") %>
    </div>
  </div>
</footer>

Homepage

It is defined in app/views/hyrax/home/_home_content.html.erb, so change that file something like:

<!-- top boxes -->
<div class="container row">
  <div class="col-xs-12 col-sm-12 ncelp-homepage-middle-box">
    <h3>Search the Resource Portal</h3>
    <%= render partial: 'homepage_search_form' %>
    Enter <b>any text</b> you wish into the search box (e.g. speaking, French, grammar) OR<br/>
    Browse by: All resources; Topic; Language; Age; Type of material</br>
    Click on "All Resources" to view all content on the Resource Portal and then use the filters to refine your search<br/>
  </div>
</div>

<!-- bottom boxes -->
<div class="container row">
  <div class="col-xs-12 col-sm-8 ncelp-homepage-bottom-box-left">
    <h4><%= t('hyrax.homepage.featured_works.tab_label') %></h4>
    <%= render 'featured_works' %>
  </div>
  <div class="col-xs-12 col-sm-4 ncelp-homepage-bottom-box-right">
    <h4><%= t('hyrax.homepage.recently_uploaded.tab_label') %></h4>
    <%= render 'recently_uploaded', recent_documents: @recent_documents %>
  </div>
</div>

Homepage search box

Can be customized in app/views/hyrax/home/_homepage_search_form.html.erb:

<%= form_tag search_form_action, method: :get, class: "form-horizontal search-form", id: "search-form-header", role: "search" do %>
  <%= render_hash_as_hidden_fields(search_state.params_for_search.except(:q, :search_field, :qt, :page, :utf8)) %>
  <%= hidden_field_tag :search_field, 'all_fields' %>
  <div class="form-group">

    <div class="input-group">
      <%= text_field_tag :q, current_search_parameters , class: "q form-control", id: "search-field-header", placeholder: t("hyrax.search.form.q.placeholder") %>

      <div class="input-group-btn">
        <button type="submit" class="btn btn-primary" id="search-submit-header">
          <%= t('hyrax.search.button.html') %>
        </button>
      </div><!-- /.input-group-btn -->
    </div><!-- /.input-group -->

  </div><!-- /.form-group -->
<% end %>

Menubar

Remove searchbox

Copy views/_controls.html.erb from Hyrax to your app, and remove this:

<div class="searchbar-right navbar-right col-sm-7">
    <%= render partial: 'catalog/search_form' %>
</div>

Install and configure zsh to be git friendly

Install zsh

Install zsh:

sudo apt-get install zsh
sudo usermod -s /bin/zsh $(whoami)

Reboot

Install zsh plugins

Install Powerlevel9k theme:

sudo apt-get install zsh-theme-powerlevel9k
echo "source /usr/share/powerlevel9k/powerlevel9k.zsh-theme" >> ~/.zshrc

Install zsh-syntax-highlighting:

sudo apt-get install zsh-syntax-highlighting
echo "source /usr/share/zsh-syntax-highlighting/zsh-syntax-highlighting.zsh" >> ~/.zshrc

Install Oh-My-ZSH:

sh -c "$(wget https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)"

Choose a theme and change .zshrc, the themes and their look can be found here:

https://github.com/robbyrussell/oh-my-zsh/wiki/themes

make sure copy all environment variables from .bashrc/.bash_profile to ~/.zshrc

Fix zsh cursor position problem:

In some version of Ubuntu, the cursor is a character behind, to fix this problem:

Create a sh script and run it:

#!/bin/sh
DIR="$HOME/.oh-my-zsh/custom/themes"
mkdir -p $DIR
cd $DIR
sed "s/➜/→/g;s/✗/×/g" $HOME/.oh-my-zsh/themes/robbyrussell.zsh-theme > robbyrussell.zsh-theme

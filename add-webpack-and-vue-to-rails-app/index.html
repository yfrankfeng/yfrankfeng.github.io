How to add webpacker and Vue.js to existing Rails application

Environments

Ubuntu 18.04.1 LTS
ruby 2.5.3p105
Rails 5.2.2

Installation

Add the following to Gemfile:

gem 'webpacker', '~> 3.5'

Run:

bundle install
bundle exec rails webpacker:install

After the above command, lots files are created/updated:

git status

It shows

modified:   .gitignore
modified:   Gemfile
modified:   Gemfile.lock
modified:   config/environments/development.rb
modified:   config/environments/production.rb
modified:   package.json      

.babelrc
.postcssrc.yml
app/javascript/
bin/webpack
bin/webpack-dev-server
config/webpack/
config/webpacker.yml
yarn.lock

The app/javascript folder:

tree app/javascript

The output is:

app/javascript
└── packs
    └── application.js

To support Vue.js, run:

bundle exec rails webpacker:install:vue

To see the files being generated and updated, run:

git status --porcelain

It shows:

 M config/webpack/environment.js
 M config/webpacker.yml
 M package.json
 M yarn.lock
?? app/javascript/app.vue
?? app/javascript/packs/hello_vue.js
?? config/webpack/loaders/

Add your own Vue files

Run:

npm install --save bootstrap-vue

Add your own files, the folder/files will be:

Run

tree app/javascript

It shows

app/javascript
├── app.vue
├── packs
│   ├── application.js
│   ├── hello_vue.js
│   └── hyrax
│       └── base
│           └── show.js
└── views
    └── hyrax
        └── base
            └── show.vue

app/javascript/packs/hyrax/base/show.js:

import Vue from 'vue/dist/vue.js';
import ResourceDetailsView from 'views/hyrax/base/show';
import BootstrapVue from 'bootstrap-vue'

Vue.use(BootstrapVue);
new Vue({
    el: '#resource-details',
    components: {
        'resource-details-view': ResourceDetailsView
    }
});

app/javascript/views/hyrax/base/show.vue

<template>
    <div class="row">
        <div class="panel panel-default">
            <div class="panel-heading">
                {{resource_show_title}}
            </div>
        </div>
    </div>
</template>

<script>
    export default {
        props: {
            resource_show_title: String
        }    
    }
</script>

<style scoped>

</style>

app/views/hyrax/base/show.html.erb

<% provide :page_title, @presenter.page_title %>

<%= render 'shared/nceheader' %>

<%= render 'shared/citations' %>

<div id="resource-details">
    <resource-details-view
      resource_show_title="Resource details">
    </resource-details-view>
</div>

<%= javascript_pack_tag 'hyrax/base/show' %>

Run following command to get Vue files compiled in Dev environment. In production, Vue will add a hook before asset precompile automatically.

bin/webpack-dev-server

Then you will be able to see the show page managed by Vue.js!
